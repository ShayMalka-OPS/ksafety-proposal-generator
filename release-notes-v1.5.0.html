VERSION 1.5.0 April 9, 2026

Hi team,

We've just released version 1.5.0 of the K-Safety Proposal Generator with a set of meaningful improvements — including bug fixes, new infrastructure flexibility, and a richer proposal experience. Here's everything that's new.

🔧 Bug Fixes

🗑️

Removed K-Analytics product line

Step 1 now shows only the three active product lines: K-Safety, K-Video, and K-Dispatch. K-Analytics has been removed from the selector.

💾

Proposal history now loads correctly

Fixed a critical bug where saved proposals were not appearing in the History page after a page reload or browser restart. All proposals stored in MongoDB Atlas are now reliably retrieved.

✨ New Features

🖥️

Full HA Mode infrastructure

Enabling High Availability Mode in Step 4 now generates a complete redundant architecture: dual Active Directory servers, dedicated application servers for each active integration (LPR, Face Recognition, Video Analytics), a 3-node Elasticsearch cluster, and a duplicate web server.

✏️

Editable VM infrastructure table

The VM infrastructure table in Step 5 (Pricing Summary) is now fully editable. You can modify server specs inline, add custom rows, delete rows you don't need, or reset the entire table back to the auto-calculated values.

💰

Discount field in Pricing Summary

A new discount percentage field in Step 5 allows you to apply a deal discount. The grand total, discounted amount, and 5-year cost comparison all update in real time.

🌍

Multi-currency display in Step 6

You can now switch the proposal preview between USD ($), NIS (₪), and MXN (MX$) with a single click. All prices in the preview convert automatically. Exports always remain in USD.

🤖

Improved AI Summary section in Step 6

Section 5 is now titled "Generative AI Summary" and the Generate button is positioned directly alongside it for a cleaner flow. Product descriptions in Section 2 now show only the modules the customer actually selected — no more generic overviews for unrelated products.

Ready to try it?

Log in to the Proposal Generator and start a new proposal to see all the new features in action. If you run into anything unexpected, reply to this email or open an issue on GitHub.
