K-Safety Proposal Generator
What's new in the latest update

Hi team,

We've just shipped a new version of the K-Safety Proposal Generator with several improvements to the proposal output, vendor support, hardware sizing, and security. Here's a quick summary.

🎨
New Proposal Design
The final proposal page has a completely new look — matching Kabatone's official template with a branded hero header, capability cards, use-case sections, and a polished pricing table. The PDF and Word exports now produce the exact same layout.
🔌
Vendor Selection per Technology
You can now specify the vendor for each technology type included in the proposal — VMS, LPR, Face Recognition, IoT, and Video Analytics. Selected vendors appear on the proposal cover band and as integration chips, clearly marking each one as Ready or In Development.
🖥️
VXG Hardware Sizing (On-Prem & Cloud)
When K1-Video is included in a proposal, the system now automatically calculates the required VXG Embedded VMS hardware — number of nodes, vCPU, RAM, and storage — for both On-Premises and Cloud (AWS) deployments. Sizing is based on camera count, bitrate, and retention days.
🔒
Secured Login — Credentials Required
Access to the tool is now restricted to registered users only. The default credentials hint has been removed from the login screen. Contact your administrator to receive your login details. Administrators can manage users and reset passwords from the Admin panel.
🐛
Bug Fix — Annual / Perpetual Pricing
A miscalculation between Annual and Perpetual unit prices when editing custom prices has been fixed. Prices now convert correctly and display the right values in both the proposal preview and the exported documents.

The update is live — just refresh your browser to get the latest version.
